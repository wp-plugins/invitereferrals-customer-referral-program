== Installation ==

This section describes how to install the plugin and get it working.

1. You can either use the automatic plugin installer or your FTP program to upload plugin files to your wp-content/plugins/invitereferrals directory. Make sure the source files reside in this directory. 
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit your InviteReferrals options (*Options - InviteReferrals*) for Wordpress 2.3.x, (*Settings - InviteReferrals*) for Wordpress 2.5.x-2.8.x 
4. Paste the key you received from http://www.InviteReferrals.com/ or get it from your www.InviteReferrals.com dashboard.
5. Test your installation by logging out of Facebook and visiting your blog.
6. Get Secret Key and brandid (used in edit module setting) as explained in the following steps
-> Go to www.invitereferrals.com and login (using information from registration)
-> Go to the Website Integration section on sidebar
-> Note the BrandID and Secret Key
